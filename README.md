Генератор случайных цитат с графическим интерфейсом, историей и фильтрацией.

## Автор
Зайнуллин Ильяс

## Описание
Это приложение на Python, использующее библиотеку `tkinter`, которое позволяет:
- Генерировать случайные цитаты из встроенной базы данных.
- Просматривать историю сгенерированных цитат.
- Фильтровать цитаты по автору и теме.
- Добавлять свои собственные цитаты в базу (с валидацией полей).
- Сохранять историю генерации в файл `quotes_history.json`.

## Технологии
- Python 3.x
- Библиотеки: `tkinter`, `random`, `json`
